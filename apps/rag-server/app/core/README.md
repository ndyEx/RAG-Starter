# 🧠 Core (파이썬 비즈니스 로직 및 서비스 계층)

이 폴더는 시스템의 핵심 뇌 역할을 하는 **비즈니스 로직(Service)** 파일들이 모여있는 곳입니다.

완벽하게 **Spring의 `@Service`** 클래스들과 역할이 같습니다.

- 컨트롤러(`api/routes`)에서 넘어온 명령을 받아,
- DB에서 데이터를 조회(`services/vectordb.py`)하고,
- 외부 AI에 분석을 요청(`services/llm.py`)한 뒤, 
- 최종 결과를 조립해서 다시 컨트롤러로 돌려주는, 흐름 제어의 중심지입니다.
